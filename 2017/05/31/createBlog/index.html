使用hexo+github创建你的个人博客

1.准备工作
在开始搭建你的个人博客之前，你必须有：
（1）一个你自己的github账号
（2）你的电脑已经安装了node.js
（3）你的电脑已经安装了git电脑客户端
2.创建仓库
新建一个“你的用户名.github.io”的仓库，比如说，如果你的github用户名是filna，那么你就新建一个filna.github.io的仓库，记住，必须用你的用户名（以后你博客的访问地址就是https://filna.github.io）。
3.配置SSH key
在你电脑的cmd中输入
$ cd ~/. ssh
如果提示：No such file or directory，那么说明你是第一次使用git
然后连续回车3次，最终会生成一个文件在用户目录下，打开用户目录，找到 .ssh\id_rsa.pub文件，记事本打开并复制里面的内容，打开你的github主页，进入个人设置，找到SSH and GPG keys,然后点击New SSH key,将刚复制的内容粘贴到key哪里，title随便填，保存。
接下来，在你的cmd中输入 $ ssh -T git@github.com,如果提示”Are you sure you want to continue connecting (yes/no)?,输入yes ,然后会看到 Hifilna!You’ve successfully authenticated,but GitHub does not provide shell access.看到这个，说明你的SSH已经配置成功。
3.使用hexo写博客
（1）安装
在你的cmd中输入$ npm install -g hexo
（2）初始化
在你的电脑的某个地方新建一个文件夹，例如我的是d:\hexo,在你的cmd中切换到该目录下，然后输入
$ hexo init
接着，输入$ hexo g生成你的博客
再输入$ hexo s启动服务器，
现在你可以打开浏览器访问http://localhost:4000，就可以看到你的网站了。
若你要修改主题，可以到hexo的官方主题下载https://hexo.io/themes/
4.上传到github
（1）打开你的hexo文件夹，修改文件夹下的_config.yml文件中有关deploy的部分：
deploy:
type: git
repository: git@github.com:filna/filna.github.io.git
branch: master
（2）安装一个插件：
npm install hexo-deployer-git –save
（3）在你的hexo文件夹中右键，打开你的git bash，输入hexo d将本次改动的所有代码全部上交。此时，你就可以在网页访问你的微博:https://filna.github.io
5.写微博
（1）在你的git bash中输入hexo new “新微博的名字”，此时你就可以在你的hexo->source->_posts文件夹下找到你新建的.md文件,将你要写的内容写在该文件中。
（2）在你的git bash中输入hexo g，再输入hexo d，刷新微博，你就可以看到它了。
