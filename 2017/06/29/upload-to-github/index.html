上传代码到github

（1）cd到你的本地根目录下（或者在你的文件夹中右键打开Git bash），执行
git init
（2）将项目的所有文件添加到仓库中
git add .
如果想添加某个特定的文件，只需把.换成特定的文件名即可
（3）将add的文件commit到仓库
git commit -m “注释语句”
（4）去github上创建自己的Resposity，获取创建仓库的https地址
（5）将本地的仓库关联到github上
git remote add origin https://github.com/filna/test.git7）
后面的https链接地址换成你自己的仓库url地址
（6）上传github之前，要先pull一下，执行如下命令：
git pull origin master –allow-unrelated-histories
（7）代码合并
git pull –rebase origin master
（8）git push -u origin master
