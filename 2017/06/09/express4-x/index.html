node+express4.x搭建网站基础

1.打开cmd，全局安装express
npm install -g express@4.x
2.安装命令工具express-generator
npm install -g express-generator
3.创建网站框架
express -e myWeb
4.进入myWeb
cd myWeb
5.初始化
npm init
6.安装依赖的模块
npm install
7.在app.js文件中添加以下代码
var index=require(“./routes/index.js”);
app.use(“/index”,index);
8.修改routes文件夹下的index.js文件如下：
var express=require(“express”);
var router=express.Router();
router.get(“/“,function(req,res,next){
res.end(“successfully”);
});
module.exports=router;
9.启动服务器
npm start
此时，在浏览器中输入http://localhost:3000/即可
